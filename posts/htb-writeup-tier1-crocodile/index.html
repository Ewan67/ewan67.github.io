┌──(ewan67㉿kali)-[~/Documents/Cybersecurity/HTB/Tier1]
└─$ gobuster dir -u http://10.129.58.242 -w /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-small.txt -x .php

Significado de las flags:

  • dir : Uses directory/file enumeration mode.
  • -u : The target URL.
  • -w : Path to the wordlist.
  • -x : File extension(s) to search for.
1
2

┌──(ewan67㉿kali)-[~/Documents/Cybersecurity/HTB/Tier1]
└─$ ffuf -u http://10.129.58.242/FUZZ -w /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-small.txt -c -t 100 -e .php -o ffuf_output -of md

Significado de las flags:

  • -u : la URL que queremos escanear.
  • -w : la ubicación del diccionario.
  • -c : para colorear la salida.
  • -t : nro de hilos concurrentes (por defecto son 40).
  • -e : listado de extensiones que queremos rasrear, separadas por comas en caso de ser más de una.
  • -o : nombre del fichero donde almacenaremos la salida.
  • -of : formato del fichero de salida, md en nuestro caso.
